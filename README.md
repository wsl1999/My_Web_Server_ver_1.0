# WEB SERVER Ver 1.0
## Run the Server
1. Run ./server in **Terminal** <u>**in the folder**</u>.
2. Log in http://localhost:8800/ in the address column.
3. Click the links on the index page to **GET** the server info.
4. Have fun exploring the server!

## Validate or Invalidate Modules
- Module list is in file "modulelist"

- use "#" to invalidate certain module(s).
